NYC, CITIBIKE DATA

Mapping the NYC Citibike Data in September 2020 and 2021

By Chuqin Jiang

June 12, 2022

This is a point map showing the start/end station of citibike in New York City in September 2020 and 2021. The size of each circle represents the number of trips starting/ending in that particular station. If you click each point, you can see the station name and the number of trips.
